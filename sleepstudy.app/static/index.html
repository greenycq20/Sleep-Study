Sleep as Android Webhook Preview

Webhook connection to log real-time snoring/coughing audio events as they occur.

Webhook Listener Ready

Setting up webhooks in Sleep as Android:

  1. Open **Sleep as Android** settings.
  2. Navigate to **Services** → **Automation** → **Webhooks**.
  3. Enable **Webhooks** and paste this URL into the endpoint field.
  4. The app will automatically send snore/cough counts during sleep tracking.
